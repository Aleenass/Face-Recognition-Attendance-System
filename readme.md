# Face Recognition Attendance System

An AI-powered attendance management system built using Python, OpenCV, Face Recognition, and SQLite.

## Features

- Face Detection
- Face Recognition
- Automatic Attendance Marking
- Automatic Absent Detection
- SQLite Database Storage
- Excel Report Export

## Technologies Used

- Python
- OpenCV
- face_recognition
- SQLite
- Pandas
- OpenPyXL

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Capture face images

python capture_faces.py

3. Train model

python train_model.py

4. Start attendance

python recognize_faces.py
